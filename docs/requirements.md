# 目的
複数のニュースサイトを並列にスクレイピングするCLIツールを作成し、
以下を学習・実践することを目的とする。

- Goにおける goroutine / context を用いた並行処理設計
- 並行処理を業務ロジックから分離する設計
- サイト追加に強いインターフェース設計

# 対象サイト
- NHKニュース
  - HTML構造が比較的シンプルで、静的スクレイピングが可能
- ITmedia
  - カテゴリ構造が明確で、サイト差異を吸収する設計練習に適している

# 取得項目
- title: 記事タイトル（string）
- url: 記事URL（string）
- published_at: 公開日時（time.Time に正規化）
- source: ニュースサイト名（NHK / ITmedia）

# CLI機能
- ニュースを並列に取得する
- サイト別件数を集計して表示する
- 最新記事を時刻順に表示する
